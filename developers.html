Information for Developers

A list of homeservers, in JSON format, is available here. You're welcomed to use it for various Matrix clients (eg. to help new users select their homeserver).

However, since this list itself is manually compiled from various raw (automatically-generated) sources, it is not advised to include the list in other raw public homeserver lists.
